HJS - 沪江前端js库¶

此js类库是沪江前端js类库的整合，HJS 即HJ和JS之和，也是我们前端类库的全局命名空间。

基本规则
  • 全局只允许HJS一个命名空间，所有模块都放在HJS下面，如：HJS.Ajax。
模块列表
  • HJS.DictSound 沪江小d单词发音功能模块
